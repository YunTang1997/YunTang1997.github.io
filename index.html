抽象类和接口的区别

发表于 2020-08-19 分类于 Java |
本文字数： 354 阅读时长 ≈ 1 分钟
  1. 接口的方法没有方法体（JDK1.8之前），JDK1.8开始也有的方法体（静态方法：使用 static关键字修饰。可以通过接口直接调用静态方法，并执行其方法体；默认方法：使用default关键字修饰。可以通过实现类对象来调用。），而抽象类可以有非抽象的方法（有方法体）。
  2. 接口的方法的访问修饰符都是public（默认public abstarct修饰），抽象类的抽象方法可以有public、protected和default这些修饰符。
  3. 抽象类中可以有普通成员变量，接口中没有普通成员变量，只有常量（默认public static final修饰）。
  4. 一个类可以实现多个接口，但只能实现一个抽象类。
  5. 抽象类可以有构造器，只是抽象类的构造器不能直接调用，但是其子类可以在构造函数中调用其父类（抽象类）的构造函数，接口没有构造器。
