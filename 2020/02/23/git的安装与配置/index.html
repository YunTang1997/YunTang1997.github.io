git的下载和安装

在Git官网下载对应版本的git软件：

然后双击安装即可，在选择Adjusting your PATH environment时选择Use Git from Git Bash only（其余默认，如果不想安装在默认路径，记得更改安装路径）。
若在桌面鼠标右键有Git GUI Here和Git Bash Here两个选项，则说明安装成功。
